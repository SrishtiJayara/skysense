# 🌤️ SkySense - Weather Forecast Website

SkySense is a real-time weather forecast web application that provides accurate weather updates using location-based services and weather APIs.

## 🚀 Live Demo
file:///C:/Users/ADITYA/Downloads/sky_sense_v2_realtime_fixed/index.html

## ✨ Features

- 🌍 Real-time location detection using GPS
- 🌦️ Current weather updates
- 🔎 Search weather by location
- 📍 Accurate latitude & longitude based data
- 🎨 Dynamic weather UI animations
- 📱 Responsive design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Open-Meteo API
- Geolocation API

## ⚙️ How It Works

The website gets the user's location coordinates, sends them to the weather API, and displays real-time weather information.
