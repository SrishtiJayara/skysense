# SkySense - Real-Time Weather App - Quick Start Guide

## 🚀 Getting Started (2 Minutes)

### 1. Open the App
- Open `index.html` in your web browser
- You'll see the welcome screen with popular cities

### 2. Get Weather (3 Options)
- **Search**: Type a city name and press Enter
- **GPS**: Click 📍 button to auto-detect your location
- **Map**: Click 🗺️ button to pick a location on the map

### 3. View Your Weather
- Current conditions with temperature and description
- Hourly forecast for the next 24 hours
- 7-day forecast
- Detailed metrics (humidity, wind, UV index, etc.)

## ⚙️ Optional: Enhanced Real-Time Features (5 Minutes)

### Get More Accurate Data
1. Click **⚙️ Settings** button (top-right)
2. Enter your **OpenWeatherMap API key** (free)
3. Click "Test Connection" to verify
4. Click "Save"

**Get your free key**: https://openweathermap.org/api

### Enable AI Chat
1. Click **⚙️ Settings**
2. Enter your **Groq API key** (free)
3. Click "Test Connection"
4. Click "Save"

**Get your free key**: https://console.groq.com

## ❓ Common Questions

**Q: Do I need an API key to use the app?**
A: No! Open-Meteo is free and doesn't require a key. API keys are optional for enhanced features.

**Q: Is my API key safe?**
A: Yes! Keys are stored only in your browser's local storage, never sent to external servers.

**Q: What if the weather data fails to load?**
A: The app automatically tries multiple providers. If one fails, it switches to another. Check your internet connection and try again.

**Q: Can I use this offline?**
A: No, real-time weather data requires an internet connection.

**Q: How often is the weather data updated?**
A: Automatically every 10 minutes. You can also manually refresh by searching for a new location.

## 🆘 Troubleshooting

### Weather won't load
1. Check internet connection
2. Try a different city
3. Refresh the page
4. Clear browser cache

### API key not working
1. Verify the key is correct
2. Check your API account is active
3. Use "Test Connection" button
4. Wait a few minutes (new keys take time to activate)

### GPS location not working
1. Allow location access when browser asks
2. Check location services are enabled
3. Try using map picker instead

### AI chat not responding
1. Ensure Groq API key is configured
2. Click "Test Connection" in settings
3. Check your Groq account has credits
